# react-native-document-opener

Opens documents with their native previewers on Android and iOS

## Installation

```sh
npm install react-native-document-opener
```

## Usage

```js
import DocumentOpener from "react-native-document-opener";

// ...

const result = await DocumentOpener.multiply(3, 7);
```
